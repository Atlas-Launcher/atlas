# Player Troubleshooting

## Readiness vs Troubleshooter

- Readiness: issues that prevent launch at all.
- Troubleshooter: issues after installation or at runtime.

## Common Signals

- Out of memory / heap errors: increase memory settings where applicable.
- Missing metadata/runtime files: run sync/repair paths.
- Java not ready: launcher runtime repair/reinstall path should recover.

## Support Bundle

Launcher support bundle includes:
- readiness state,
- recent logs/status,
- structured findings,
- redacted sensitive values.

Use this bundle when reporting issues.
