# Player overview

This section is for players who want the fastest path from invite to play.

## Read this section in order

Use this order to move from setup to recovery without repeating steps.

1. `getting-started.md`: Complete first-time setup and first launch.
2. `quick-reference.md`: Use a short day-to-day checklist.
3. `troubleshooting.md`: Resolve readiness and launch failures.

## What this section covers

These guides focus on launcher actions you can complete without CLI tooling.

- Invite acceptance and launcher handoff.
- Account readiness through Launch Assist.
- First launch, retries, and support bundle workflow.
